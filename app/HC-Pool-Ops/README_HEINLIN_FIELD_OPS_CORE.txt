HEINLIN FIELD OPS CORE BUILD

Install:
1. Extract this ZIP.
2. Copy everything inside it into your HC-Pool-Ops-FIXED-BY-JARVIS folder.
3. Choose Replace All.
4. Run: .\INSTALL_HEINLIN_FIELD_OPS_CORE.bat
5. Start server: uvicorn app.app:app --host 0.0.0.0 --port 8000 --reload
6. Login: mike / mike

This is not the final dream app. It is a stable core build that actually creates edit screens and saves photos/data.
